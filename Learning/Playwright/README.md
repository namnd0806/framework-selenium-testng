## Tại sao học Playwright?

```
2022-2023: Selenium vẫn là king
2024-2025: Playwright tăng trưởng mạnh, nhiều dự án mới chọn Playwright
2026+:     Playwright và Selenium song song tồn tại

Biết cả 2 = lợi thế cạnh tranh lớn
```
