> Dành cho người đã biết Selenium. Appium dùng cùng WebDriver API nhưng cho mobile.

## Lộ trình học

```
Hiểu mobile app types (Native/Hybrid/Web)
    ↓
Setup môi trường (Android SDK, Appium Server)
    ↓
Tìm locator với Appium Inspector
    ↓
Viết test cơ bản (tap, type, swipe)
    ↓
Page Object Model cho mobile
    ↓
Parallel + Cloud testing
```

## Prerequisite

- Selenium cơ bản (cùng WebDriver API)
- Java cơ bản
- Android Studio (cho Android testing)
