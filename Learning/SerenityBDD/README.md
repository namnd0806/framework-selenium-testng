## Serenity vs Cucumber thuần — Khi nào dùng cái nào?

| | Cucumber thuần | Serenity BDD |
|---|---|---|
| Setup | Đơn giản hơn | Phức tạp hơn |
| Report | Cần plugin riêng | Built-in, rất đẹp |
| Living docs | Không có | Có — requirements hierarchy |
| @Steps | Không có | Có — nested steps trong report |
| Screenplay | Không có | Có |
| Dùng khi | Project nhỏ, team mới | Enterprise, cần report đẹp |
