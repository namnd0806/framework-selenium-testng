## Tips học tiếng Anh IT

1. **Đọc tài liệu gốc:** Selenium docs, TestNG docs, Stack Overflow
2. **Viết commit message bằng tiếng Anh** (Conventional Commits)
3. **Comment code bằng tiếng Anh**
4. **Tham gia cộng đồng:** GitHub issues, Stack Overflow
5. **Xem video tiếng Anh:** Testing conferences (SeleniumConf, TestBash)
